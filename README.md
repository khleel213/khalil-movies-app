🎬 Eng. Khalil - Movies Explorer
A modern web application built to search for movies and display their details using real-time data.

🔗 Live Demo
Check out the live project here: https://khleel213.github.io/khalil-movies-app/

✨ Key Features
Instant Search: Find any movie and get its data immediately.

Professional Display: Clean layout for movie posters, ratings, and release years.

Responsive Design: Optimized for both desktop and mobile screens.

Live Data: Fetches real-time information using the IMDB API.

🛠️ Tech Stack
HTML5: For the application structure.

CSS3: For styling, Flexbox, and responsive layouts.

JavaScript (ES6): For logic and handling API data using Async/Await.

GitHub Pages: For hosting and online deployment.

📂 Project Structure
index.html: The main user interface.

style.css: All design and color configurations.

script.js: The core logic for API integration and UI updates.
